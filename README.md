プロットする時は`file_pass`と`file_name`の設定をしてからプログラムを実行してください．

`csv_plot.py`：
一変数関数をプロット

`csv_plot_3d.py`：
二変数関数をプロット

例として `plot_test_2d.csv` と `plot_test_3d.csv` を用意している．
